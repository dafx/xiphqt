Resource: scalelenadj

applicability

Toplevel, Module class or instance

possible settings

0<=n<=100
(default varies my module, 100 is global default)

description

'scalelenadj' sets the length of a module's bargraph to a percentage relative to the height/width of the main window depending on stacking. When the graph bars and stacking are in the same direction, the scalelenadj values for all the active modules are summed and each module is allotted a length proportional to its scalelenadj value plus it's minimum size value (see the geometry page for a detailed discussion of space allocation).

examples

cpustat is a module that uses a justification of 1 and a scalelenadj setting of 100. loadav uses a scalelenadj of 60 and a justification of 1. phoon uses a scalelenadj of 60 and a justification of 0.
