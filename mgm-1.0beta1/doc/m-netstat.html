Module: netstat

Displays current network throughput broken down by interface. Displays bytes/second when possible, otherwise packets/second. The module will automatically reconfigure when root adds or removes interfaces.

Current platforms

Linux

Instances

single only

additional resources

none

Resource defaults

Resource defaults explicitly altered from the global defaults by this module:
  • bar.n.label: if Rx
  • bar.n+1.label: if Tx
  • bar.n.litbackground: #787cf8
  • bar.n+1.litbackground: #78b0f8
  • order: 20
  • scalereturn: 120
  • scalewidadj: 160*if

Possible enhancements

Could (and should) be extended to use vmstat/iostat utilities on platforms other than Linux (eg, Solaris *BSD).
