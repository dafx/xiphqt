Resource: lendemand

applicability

Module class or instance

possible settings

>0 (25 is default)

description

lendemand is used along with widdemand and the per-module scalelenadj and scalewidadj to determine the actual requested size of the MGM window when a geometry is not explicitly set. See the geometry page for a description of how these resources interact to determine overall window geometry and sizing/placement of individual modules.

lendemand specifically maps the global scale length adjustment, a relative number, to an actual number of pixels; the global length adjustment is multiplied by the lendemand setting and divided by 100. The result is then used as the desired window height for a an MGM window with vertical bars or the widow width for an MGM window with horizontal bars.
