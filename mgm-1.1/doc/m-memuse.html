Module: memuse

Displays current memory/swap usage. It does not count buffers or the page cache as 'used'.

Current platforms

Linux

Instances

single only

additional resources

none

Note that the refresh frequency for this module should not be set too high at the moment; reading /proc/meminfo in Linux 2.0 through current 2.3 is much more expensive than you'd expect. The 'mongo low latency' pathes for 2.3 correct this problem.

Resource defaults

Resource defaults explicitly altered from the global defaults by this module:
  • bar.0.label: memory (total)
  • bar.1.label: swap (total)
  • litbackground: #e8d088
  • order: 101
  • scalerefresh: 2000
  • scalewidadj: 160

Possible enhancements

Could (and should) be extended to use vmstat/iostat utilities on platforms other than Linux (eg, Solaris *BSD).
