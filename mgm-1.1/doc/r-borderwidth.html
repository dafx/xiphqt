Resource: borderwidth

applicability

Toplevel, Module class or instance

possible settings

>=0 (0 default)

description

Borderwidth is used to set the relief border width on modules that support a relief (such as phoon, clock, hostlogo and spacer). For other modules it has no effect. Note that the graph bar borderwidths are not set by this resource (they're hardwired).
