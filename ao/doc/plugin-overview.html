libao Plugin Overview

Plugins are drivers that are loaded dynamically when libao is first initialized by the client application. Drivers that are operating system dependent, like the oss and sun drivers, or that depend on external libraries, like the esd driver, must be implemented as plugins in order to keep binary packagers happy.
