Module: diskstat

Displays disk read/write throughput by disk in kB/sec.

Current platforms

Linux

Instances

single only

additional resources

none

Resource defaults

Resource defaults explicitly altered from the global defaults by this module:
  • label: diskN read/write(where N is disk number)
  • litbackground: #e7ad74 (read) or #ade774 (write)
  • order: 10
  • scalereturn: 120
  • scalewidadj: 150*disks

Possible enhancements

Could (and should) be extended to use vmstat/iostat utilities on platforms other than Linux (eg, Solaris *BSD). Also, currently renumbers disks into a contiguous order on Linux; shouldn't do that.
