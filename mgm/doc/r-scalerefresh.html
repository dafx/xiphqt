Resource: scalerefresh

applicability

Toplevel, Module class or instance

possible settings

>0
(in milliseconds) (default varies my module, 500 is the global default)

description

scalerefresh sets a module's refresh rate. The module's 'module_update' method is called once every scalerefresh milliseconds.

see also
