Module: spacer

Draws a spacer oriented the same direction as the graph bars.

Current platforms

All

Instances

multiple allowed

Additional resources

thickness

thickness sets the pixel width of the spacer.

Resource defaults

Resource defaults explicitly altered from the global defaults by this module:
  • background: #27536c
  • borderwidth: 1
  • thickness: 1
  • order: 99,199
  • relief: sunken
  • scalejustify: 0
  • scalelenadj: 50
  • scalewidadj: 20
This is the only module that creates multiple instances by default, using an 'order' setting of '99,199'.

Possible enhancements

The rendering direction should probably be set by stack direction, not bar direction... also, taking an XPM instead of only drawing a relief might be nice.

MGM will not get your whites whiter or your colors brighter. It will, however, sit there and look spiffy while sucking down a major honking wad of RAM.
