Resource: labelsize

applicability

Toplevel, Module class or instance

possible settings

>0 (9 default)

description

The 'labelsize' resource sets the size of the scale range label text on bargraphs. The given number is the height of the text in pixels (width is scaled proportionally). This only affects scalelabels; the size of text in modules that just use standard X fonts is inherent in the font name. Text in bargraphs bars is scaled to be as high as the bar is wide, then compressed if necessary to fit lengthwise (the aspect ratio of bar text can be manipulated with ratio).
