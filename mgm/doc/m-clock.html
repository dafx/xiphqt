Module: clock

This, essentially, is xclock in a convenient plugin form for those who don't also want to run an external xclock.

Current platforms

All

Instances

multiple allowed

additional resources

ipad, litbackground

ipad specifies an internal padding to add to the inside of all four module borders. The padding is added inside any relief.

litbackground is a standard resource, but clock maps it a little strangely; here, litbackground specifies the color of the clock face tic-marks.

Resource defaults

Resource defaults explicitly altered from the global defaults by this module:
  • borderwidth: 1
  • ipad: 20
  • order: .1
  • relief: sunken
  • scalejustify: 0
  • scalelenadj: 60
  • scalerefresh: 2000
  • scalewidadj: 150

Possible enhancements

The text label that reads the day of the week and 24 hour time should be modifiable/disableable via 'label' like the label in hostname. Right now it's hardwired.
