Resource: reconfig

applicability

Toplevel, Module class or instance

possible settings

'true' (default), 'false'

description

A number of MGM modules (eg, diskuse and netstat) will notice when graphs must be added or removed due to mounting/unmounting filesystems, adding/removing network interfaces, etc. The module asks MGM to rerender the main window with the new configuration.

'reconfig' may be set to 'false' to disable automatic reconfiguration.
