Resource: label

applicability

Toplevel, Module class or instance, individual window, individual bar

possible settings

any text string

description

The 'label' resource sets text labels across all classes of widgets.

For example, hostlogo uses it to set the default text hostname label (which may be overridden). The bargraph modules set 'label' for each bar widget (this may also be overridden by the user). See module documentation for any special uses of label.

The following example sets the text in bar 0 of the 'netstat' module to 'foo': 

mgm*Netstat.bar.0.label:   foo

The following example sets the text in bar 1 of the first module to 'bar': 

mgm.0.bar.1.label:         bar
