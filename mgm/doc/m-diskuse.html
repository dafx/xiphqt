Module: diskuse

Displays percentage usage of mounted filesystems. Ignores 'proc' and 'AFS'. The module will automatically reconfigure if filesystems are mounted or unmounted.

Current platforms

All (requires df; only tested on BSD/Solaris/Linux)

Instances

single only

additional resources

none

Resource defaults

Resource defaults explicitly altered from the global defaults by this module:
  • label: (device or fs label)
  • litbackground: #e8d088
  • order: 100
  • scalerefresh: 30000
  • scalewidadj: 80*filesystems
