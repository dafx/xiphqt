Resource: scalethresh

applicability

Toplevel, Module class or instance

possible settings

>0 (represents an amount of the scale as 1/n)
(default varies my module; global default is 4)

description

The scalethresh specifies the graph display level under which the graph should reduce the scale range.

If, during the past (scalerefresh * scalereturn) milliseconds period, no graph value exceeded 1/scalethresh of the current scale range, the scale range will be halved. This assumes, of course, that the given module is using autoranging graphs.

see also
