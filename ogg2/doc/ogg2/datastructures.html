Base Data Structures

Libogg uses several data structures to hold data and state information.

All the libogg specific data structures are declared in "ogg2/ogg.h".

datatype purpose
ogg_page This structure references the data of an Ogg bitstream page.
ogg_packet This structure references the data and metadata for an Ogg packet.
